Sistem Inventory
Sparepart Motor

2024–2025

Sistem manajemen inventori digital untuk Bengkel Jaya Motor — mengotomasi seluruh alur pengelolaan stok sparepart yang sebelumnya dilakukan secara manual dan rentan kesalahan.

Dibangun selama program magang, aplikasi ini menjadi solusi nyata bagi pelaku usaha kecil untuk meningkatkan akurasi data dan efisiensi operasional.

React.js Vite Tailwind CSS MySQL JavaScript
Fitur Utama
Monitoring stok real-time dengan Low Stock Alert (<5 unit)
Pencatatan barang masuk dari pemasok secara terstruktur
Transaksi penjualan dengan pengurangan stok otomatis
Koreksi stok dengan audit trail lengkap
Manajemen katalog sparepart (CRUD)
Laporan stok, barang masuk & keluar
